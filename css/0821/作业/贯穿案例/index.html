代办过户服务

  • 标准代办过户服务：包含车辆外检、过户出票、过户资料递交、车辆档案提取等全程代办服务
  • 绿色过户通道：提供专用过户窗口，无需排队取号
  • 快速验车通道：提供专用验车通道，无需排队取号
  • 免费停车
  • 客休区：过户环节提供舒适的茶歇场地

2年4万公里售后保障

         交易完成后 2年4万公里 全车13大系统售后保障，累计赔付总额上限达4.55万元。

售后保障申请流程

  1.    您只需拨打瓜子二手车客服专线  400-066-8780，反映车辆问题及相关信息。
  2.    请您携带相关证照等资料，将车辆送至指定维修商处检查车辆。
  3.    瓜子二手车及指定维修商会为您的爱车确定车辆问题及相关费用。
  4.    维修完成，客户提车，瓜子二手车按约定承担相应费用。

           调表车赔付

         经瓜子二手车平台成交的二手车为调表车，并且瓜子未尽到信息披露义务的，瓜子将退还客户服务费并进行损失赔付。

     调表车定义：
卖家通过各种手段更改车辆表显里程数，使表显里程数小于实际里程数的车辆
     赔偿标准：
除全额退还客户服务费外，瓜子还将赔付买家车辆调表前后的市场价差，该市场价差瓜子将综合参考第三方平台同等年份、同等车型车辆报价和实际市场成交价决定
     赔偿规则：
经瓜子二手车直卖网成交的二手车，如在交易过户后30个自然日内（含）提供合法有效的证据证明交易车辆为调表车的，瓜子将进行损失赔付
证据备件：包含并不限于能证明该车为调表车的车辆保养记录、行车电脑里程数据等
