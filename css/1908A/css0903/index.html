习近平5年遍访57国 东风拂面世界新

这是中国元首外交精彩纷呈的5年这是为人民谋福，为国家负责，为世界担当的5年。3013年3月22日，习近平就任国家主席后首次出访，5年来，遍访世界57个国家，接待了110多为外国元首访华。运筹帷幄，亲力亲为，习近平主席引领中国特色大国外交不断前行。中国地位显著提升，中国角色备受瞩目，中国理念大放异彩。

台湾各界热议习主席讲话：反分裂意志坚定有力

中共中央总书记，国家主席，中央军委主席习近平在十三届全国人大一次会议闭幕会上发表的重要讲话，在台湾引起强烈反响。岛内各界人士的舆论表示，习主席在讲话中国对两岸关系的论述具有重大意义，体现了大陆方面与台湾同胞分享发展机遇的诚意善意，也表明大陆方面具有维护国家主权和领土完整，实现祖国完全统一的意志，信心和能力，发出了反对“台独”，反对一切分裂祖国行径和伎俩的历史强音。
